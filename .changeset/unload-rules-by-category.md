Set item unload rules for a whole category at once.

The Individual Item Unload Settings window can now apply a rule to an entire category in one action instead of going item by item. Open a category and use "Set whole category…" to make everything under it never unload, keep at most a set amount, always unload, or return to default. That turns setting up a large group like a combat mod's many ammo types into a single click. The list also covers every item in the game and your installed mods, organised by category the way a workbench's bill filter is.

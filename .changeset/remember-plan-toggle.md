Reuse your last route plan in one click.

A new "Remember plan" toggle sits with the other interface toggles in the bottom-right corner, and there is a matching "Remember plan" checkbox in the plan window itself, so it is easy to find. It is on by default. With it on, choosing "Plan prioritized …" reuses the settings you last used right away without opening the planner window, and the right-click option reads "(remembered)" so you know it will; turn it off and the planner opens each time, as before. The first time you plan a route for a new kind of target it still opens the window, since there is nothing to reuse yet. A remembered click follows the usual queued-order rules: a plain click replaces the current job and Shift-clicking appends to the queue, the same as shift-clicking any other prioritized order. Hovering the option or the checkbox blinks the interface toggle so you can see they are the same setting. The harvesting and mining routes remember their settings per target type; the sowing and floor-removal routes reuse the last plan you made.
